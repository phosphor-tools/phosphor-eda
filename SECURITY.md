# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities privately through GitHub Security
Advisories rather than opening a public issue.

Use the **Report a vulnerability** button under the
[Security tab](https://github.com/phosphor-tools/phosphor-eda/security/advisories/new)
of the repository. This opens a private advisory visible only to the
maintainers.

Please include:

- a description of the vulnerability and its impact,
- steps to reproduce or a proof of concept, and
- any affected versions you are aware of.

We will acknowledge your report, investigate, and coordinate a fix and
disclosure timeline with you. Please give us a reasonable opportunity to
address the issue before any public disclosure.
